# MafiaScum User Bot

A simple automated bot account for mafiascum.net, I wish to create a simple proof of concept for the following behaviour:

- [x] Persistently logged in, log back in when you are not
- [x] Posting a message in specific threads
- [ ] Editing an old message
- [ ] Sending PMs to specific user/s

The behaviour listed above, even if it checked, has not been extensibly tested
